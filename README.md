# Predictive Parking & Automated Dispatch System

## Overview
This repository contains the **Predictive Parking & Automated Dispatch System**, developed to address the issue of parking-induced congestion and poor operational visibility on city streets. By transforming raw historical police citation logs into actionable spatiotemporal intelligence, this system models an optimized resource dispatch plan for preemptive traffic enforcement.

## Features
1. **Spatiotemporal Data Engineering**: Extracts features and clusters parking violations using DBSCAN (Haversine metric) to pinpoint physical hotspots.
2. **Impact Calculation**: Quantifies the true impact of a hotspot based on the volume of historical violations and variance across multiple weeks (confidence score) to deduce estimated "Commuter Minutes Lost".
3. **Automated Dispatch Knapsack Optimizer**: Uses the `pulp` library to solve the Integer Linear Programming (ILP) problem. It dynamically allocates a finite number of officers to hotspots to maximize overall impact relief.
4. **Command Center Dashboard**: An interactive UI built on Streamlit with Folium for tactical mapping and dispatch manifests.

## Files
- `process_data.py`: The data pipeline script that ingests raw citations, clusters the coordinates, aggregates intensity, and produces the output matrix.
- `app.py`: The interactive Streamlit dashboard application that visualizes hotspots and handles real-time resource optimization.
- `unified_friction_log.csv`: The processed spatiotemporal matrix generated by the pipeline, mapping hotspots, delays, and resource costs.

## Installation & Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/Harshitmishra001/Parking_congestion_hotspot_analysis.git
   ```
2. Install the required dependencies:
   ```bash
   pip install streamlit pandas pulp folium streamlit-folium scikit-learn numpy
   ```
3. *(Optional)* Run the data engineering pipeline (requires the raw `violation_anonymized.csv` in the root folder):
   ```bash
   python process_data.py
   ```
4. Launch the Command Center Dashboard:
   ```bash
   streamlit run app.py
   ```
